HTML태그


[strong] 글자의 두께를 두껍게 할때 사용하는 태그

[u] 글자의 밑줄을 만들때 사용하는 태그

[br] 글자에 공간을 만들때 사용하는 태그

[p] 달락을 나눌때 사용하는 태그

[img scr="~"] 이미지를 넣을때 사용하는 태그

[li] 목차를 만들때 사용하는 태그

[ul] 목차와 목차 사이의 경계를 만들때 사용하는 태그

[ol] 자동으로 넘버링을 할때 사용하는 태그

[title] 사이트에 제목을 설정할때 사용하는 태그

[head] 본문을 설명하는 글을 감싸는 태그

[body] 분문을 감싸는 태그

[html] 모든 코드를 감싸는 태그
